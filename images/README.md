# Food Images

## How to Add Images:

1. Download images for each dish from Google Images or stock photo sites
2. Save them in this folder with these exact names:
   - `jeera-rice.jpg`
   - `chicken-thali.jpg`
   - `mutton-thali.jpg`
   - `veg-thali.jpg`
   - `biryani.jpg`
   - `paneer-tikka.jpg`
   - `dal-makhani.jpg`
   - `butter-chicken.jpg`
   - `gulab-jamun.jpg`
   - `rasgulla.jpg`
   - `lassi.jpg`
   - `masala-chai.jpg`

## Free Image Sources:

- Unsplash: https://unsplash.com/s/photos/indian-food
- Pexels: https://www.pexels.com/search/indian%20food/
- Pixabay: https://pixabay.com/images/search/indian%20food/

## Image Requirements:

- Format: JPG or PNG
- Size: 800x600px recommended
- File size: Under 500KB for fast loading
